# Cinema — веб-приложение кинотеатра на Django

Проект по вариантам в рамках курса backend-разработки.  
Вариант 12: Кинотеатр
Приложение реализует работу кинотеатра: просмотр афиши, расписания сеансов и покупку билетов.

---

## Описание проекта

Веб-приложение позволяет:

- просматривать список фильмов с описанием
- видеть расписание сеансов по залам
- выбирать места в зале
- покупать билеты
- видеть занятые места
- получать уведомление об успешной покупке билета

Проект реализован с использованием **Django** и **HTML-шаблонов**.

---

## Используемые технологии

- Python 3.10+
- Django 4.x
- SQLite
- HTML / CSS
- Django Templates

---

## Запуск

- `python -m venv venv`
- `venv\Scripts\activate`
- `pip install -r requirements.txt`
- `python manage.py migrate`
- `python manage.py createsuperuser`
- `python manage.py runserver`
